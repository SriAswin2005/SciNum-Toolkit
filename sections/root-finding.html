Root Finding Methods

Find roots of equations using various numerical methods

Fixed Point Iteration

Formula: xk+1 = f(xk)

Newton-Raphson Method

Formula: xk+1 = xk - f(xk)/f'(xk)

Bisection Method

Halves interval [a,b] where f(a)f(b) < 0

Secant Method

Formula: xk+1 = xk - f(xk)(xk-xk-1)/(f(xk)-f(xk-1))
